# AI Notes Search Assistant

A simple Python-based note retrieval system.

## Features
- Load notes from a text file
- Convert notes into a dictionary
- Search notes by topic
- Case-insensitive search
- Interactive command-line interface

## Technologies
- Python
- File Handling
- Dictionaries
- Functions
- Exception Handling
